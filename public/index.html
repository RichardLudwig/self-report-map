What is a Self-Report Survey?

In sociology, self-report surveys are a type of survey where you can repond free from interference from researchers. Use the map below to publish instances of crime where you live so that these instances can be documented and help others avoid future victimization.

I have also added navigation features so that you can choose safer routes, based on the information collected to date.

  • To add a report, click the "Add Report" button.
  • To learn more about the FBI's Uniform Crime Reporting Program, as well as the definitions for each crime recorded by this mapping application, click the "What is the UCR?" button (opens in new window).
  • To view more public safety applications, click on the link to my website in the footer.
